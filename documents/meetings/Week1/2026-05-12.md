## ✅ 결정 사항

- 도메인 역할을 다음과 같이 분담했다.
    - 회원 / 판매자 / 인증 : 임호진
    - 상품 / 카테고리 / 검색 : 이중현
    - 주문 / 결제 / 쿠폰 / 포인트 : 정지훈
    - 배송 / 리뷰 / 구독 : 김예은
    - 관리자 / 공통 인프라(CI/CD, Security, 모니터링) / AI : 정은지

- MVP 기능을 다음과 같이 확정했다.
    - 회원가입 / 로그인
    - 상품 등록 / 조회
    - 장바구니
    - 주문 / 결제
    - 배송 상태 변경
    - 관리자 상품 승인 및 반려

- 리뷰, 포인트, 쿠폰, 구독, 인기 검색어는 MVP 이후 단계에서 구현하기로 했다.

- AI 쇼핑 어시스턴트, AI 리뷰 요약, Kafka, Rate Limiting은 확장 기능으로 분류했다.

- 배송 상태값을 ACCEPT → INSTRUCT → DEPARTURE → DELIVERING → FINAL_DELIVERY로 정의했다.

- 비회원 장바구니는 Redis를 활용하고 로그인 시 DB로 이전하는 방식으로 구현하기로 했다.

- 결제 기능은 Mock 기반 2단계 플로우로 구현하기로 했다.
