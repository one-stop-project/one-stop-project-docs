## 리뷰

- 리뷰 작성 조건
    - 배송 완료(FINAL_DELIVERY) 후에만 리뷰 작성 가능
    - 본인이 구매한 주문에 대해서만 리뷰 작성 가능
    - 리뷰 작성 가능 목록은 FINAL_DELIVERY 완료 + 리뷰 미작성 주문 상품만 조회
    - CANCELLED / REJECTED 주문은 리뷰 작성 불가
- 리뷰 작성 제한
    - 주문 상품 1 종류 당 리뷰 1개만 작성 가능 (orderItemId)
    - 같은 주문 상품에 대해 중복 리뷰 작성 불가
- 별점:
    - 1점 ~ 5점
- 리뷰 내용:
    - 최소 10자
    - 최대 1,000자
- 리뷰 이미지
    - 최대 5장까지 등록
- 작성 후 30일 이후 수정 불가
- 본인 리뷰만 삭제 가능
    - 리뷰 삭제는 soft delete(ACTIVE → DELETED) 처리
- 리뷰 조회
    - 마이페이지에서 내가 작성한 리뷰 목록 조회 가능 (`GET /api/users/me/reviews`)
    - 판매자는 본인 상품의 리뷰를 전체/상품별/요약으로 조회 가능 (`GET /api/seller/reviews`, `GET /api/seller/products/{productId}/reviews`, `GET /api/seller/reviews/summary`)
    - 구매자/공개용 "상품 상세 페이지 상품별 리뷰 목록 조회" API는 현재 미구현 (상품별 리뷰 조회는 SELLER 전용)
